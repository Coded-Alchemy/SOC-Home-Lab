# SOC Analysis Tools Quick Reference

| Tool                | Use Case                                   | Link                         |
|---------------------|--------------------------------------------|------------------------------|
| VirusTotal          | File hashes, IPs, URLs, domains            | virustotal.com               |
| Talos Intelligence  | IP/URL reputation                          | talosintelligence.com        |
| IPVoid              | IP blacklist check                         | ipvoid.com                   |
| URLVoid             | URL reputation                             | urlvoid.com                  |
| AbuseIPDB           | IP abuse reports                           | abuseipdb.com                |
| Any.run             | Interactive malware sandbox                | any.run                      |
| Joe Sandbox         | Deep file/URL analysis                     | joesandbox.com               |
| Hybrid Analysis     | File/URL sandbox                           | hybrid-analysis.com          |
| DomainTools (Whois) | Registrant / domain info                   | whois.domaintools.com        |
| URLScan.io          | Website snapshot & reputation              | urlscan.io                   |
| Archive.org         | Historical website snapshots               | web.archive.org              |
| WhereGoes           | URL redirect chain analysis                | wheregoes.com                |
| Spamhaus            | IP/domain blocklist                        | spamhaus.org                 |
| IBM X-Force         | IoC intelligence                           | exchange.xforce.ibmcloud.com |
| TOR Exit Node List  | Check if IP is a TOR exit node             | dan.me.uk/torlist            |
| Google              | Context, sandbox reports, researcher blogs | google.com                   |

> **Reminder:** Always defang URLs and IPs in your analysis to prevent accidental clicks.
> Format: `www[.]example[.]com` | `hxxps://malicious[.]site` | `192[.]168[.]1[.]1`
